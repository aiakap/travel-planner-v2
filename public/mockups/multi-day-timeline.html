Design Notes - Option C:
  • Multi-day items displayed as prominent "sticky" cards at the top
  • Each card has an embedded mini-timeline showing the span

How to iterate:

  1. Open browser DevTools (F12 or Cmd+Option+I)
  2. Use the Elements panel to edit HTML and see changes live
  3. Adjust Tailwind classes to tweak spacing, colors, sizing
  4. When satisfied, copy the patterns to the React components
