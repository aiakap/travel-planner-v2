## Accessibility Features

- **Keyboard Navigation:** Tab through suggestions, Enter to accept, Delete to reject
- **Screen Readers:** Clear labels for all buttons and actions
- **Color Contrast:** All text meets WCAG AA standards
- **Focus Indicators:** Visible focus states on all interactive elements
- **ARIA Labels:** Descriptive labels for dynamic content

## Animation Timing

- **Suggestion Appear:** 200ms fade-in
- **Suggestion Accept:** 300ms fade-out
- **Suggestion Reject:** 150ms fade-out (faster)
- **Graph Node Appear:** 400ms scale-in with bounce
- **Category Appear:** 500ms fade + scale
- **Loading Spinner:** Continuous rotation
